# 修改版 IEEE 39 节点完整算例数据（极热无风场景）

本文档给出基于 **MATPOWER case39**（New England 10 机 39 节点系统）修改后的**完整算例数据**，涵盖 39 个节点、46 条支路、10 台机组（火电/水电/风电/光伏四类）、21 个负荷节点及极热无风气象场景参数。

功率单位 **MW**，基准容量 $S_B = 100\ \text{MVA}$。MATLAB 数据文件：`matlab/case39_ehnw.m`、`matlab/weather_scenario.m`。

---

## 1. 系统概况

| 项目 | 数值 |
|---|---|
| 算例名称 | case39_ehnw（Extreme Heat / No Wind） |
| 节点数 | 39 |
| 支路数 | 46 |

| 负荷节点数 | 21 |
| 基准负荷合计 $P_{D}^0$ | 6254.23 MW |

| 平衡节点 | bus 31（G2，火电-燃煤） |

### 1.1 相对原始 case39 的主要修改

| 修改项 | 说明 |
|---|---|
| 机组类型 | 原 10 台核电/火电/水电重新指派为四类电源 |
| 额定容量 | 按四类机组重新设定 $P^{\mathrm{rated}}$ |
| 发电成本 | 设二次成本 $C=c_2 P^2 + c_1 P$（$/h） |
| 运行参数 | 新增 `genOps`：基准出力 $P^0$、爬坡率、水电电量系数 |
| 负荷模型 | 保留原始 $P_{D}^0$，供温度修正与三级切负荷拆分 |
